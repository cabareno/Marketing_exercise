## Descripción

Ejercicio en el que se procesa un archivo plano proveniente de múltiples fuentes de publicidad.

## Procesamiento de Datos

Se realiza la identificación y eliminación de columnas inconsistentes o “fantasma”, así como la depuración de formatos y corrección de valores que no correspondían a la estructura esperada de la base.

## Cálculo de Métricas

Posteriormente, se calculan métricas como ROAS y efectividad de campaña.

## Resultado

Se genera una base de datos limpia y estructurada, lista para consumo por usuarios finales o para consulta.
